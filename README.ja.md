# vr-drums
English README: [README.md](README.md)

バーチャルリアリティ (VR) ドラミング体験。

## 機能
- 実在するドラムモデルを使った VR ドラミング体験
- デスクトップと VR (Quest) の両プラットフォームをサポート
- スペースのシーンがバックグラウンドにある

## 使用法
プロジェクトを実行するには、Webブラウザで`index.html`ファイルを開きます。VRヘッドセットを使用している場合、VRモードに自動的に適応します。
